## What is TailAdmin?

TailAdmin is a free and open-source admin dashboard template built on **Next.js 16** and **Tailwind CSS v4**. It provides everything you need to create a feature-rich, data-driven back-end, dashboard, or admin panel for any web project.

With TailAdmin you get access to:

- A complete ecommerce dashboard with metrics, charts, tables, and maps
- 30+ pre-built UI components (buttons, alerts, modals, tables, charts and more)
- Authentication pages (sign in and sign up)
- Calendar, profile, and utility pages
- Dark mode support out of the box
- TypeScript-first codebase built on React 19

## Tech stack

| Technology | Version | Role |
|---|---|---|
| Next.js | 16.x | App Router, SSR, SSG, API routes |
| React | 19 | UI rendering |
| TypeScript | 5.x | Type safety |
| Tailwind CSS | v4 | Utility-first styling |
| ApexCharts | 4.x | Line and bar charts |
| FullCalendar | 6.x | Calendar component |
| flatpickr | 4.x | Date picker |
